## Objective

Build the left-pane widget that displays PRs organized into collapsible sections by query group, with summary information for each PR row.

## Requirements

1. Collapsible sections, one per query group
2. Section headers show group name and PR count
3. PR rows display: title (truncated), author, age, CI status icon, review status icon
4. Emit a selection event when a PR row is highlighted/selected
5. Replace the left-pane placeholder from T05

### PR Row Display Format

Each PR row should show:
```
[CI] [Review] Title (truncated)...        author  2d
```

Status icons:
- CI: `*` passing, `!` failing, `~` pending, ` ` none
- Review: `+` approved, `x` changes requested, `?` pending, ` ` none

### Section Headers

```
[v] Requested Reviewer (3)
    [*][?] Fix auth token refresh...    alice  2d
    [!][x] Update CI pipeline...        bob    5h
    [*][+] Add user settings...         carol  1d
[>] Team Reviewer (0)
[v] My PRs (1)
    [~][?] WIP: New feature...          me     30m
```

- `[v]` = expanded, `[>]` = collapsed
- Group header is clickable/selectable to toggle collapse

### Selection Behavior

- Cursor highlights the current PR row
- Moving the cursor to a new PR emits `PRSelected` with the PR data
- Groups with 0 PRs show the header but no rows
