## Objective

Add Textual's built-in command palette with a custom `PRDashCommandProvider` exposing app commands (jump to group, refresh, settings, query groups, change theme, sort PRs).

## Background

Power users benefit from a fuzzy-searchable command palette (similar to VS Code's Ctrl+P) for fast access to all app actions without memorizing keybindings. Textual has a built-in command palette system that supports custom `Provider` subclasses yielding `Hit` objects for fuzzy matching.

## Requirements

1. **Command provider**: Create a `PRDashCommandProvider` subclass of `Provider` in `app.py`
2. **Search method**: Override `async def search(self, query: str)` to yield `Hit` objects for each available command
3. **Commands exposed**: Jump to group (one per configured group), Refresh, Open Settings, Change Theme, Sort PRs
4. **Registration**: Set `COMMANDS` class variable on the `App` class to include `PRDashCommandProvider`
5. **Keybinding**: Ensure Ctrl+P activates the palette (Textual default)
6. **Tests**: Verify provider yields expected hits, test that palette activates via keybinding
