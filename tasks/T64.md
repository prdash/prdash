## Objective

Show an "auto-merge enabled" badge via the `autoMergeRequest` GraphQL field and provide a keybinding to toggle auto-merge on/off.

## Background

GitHub's auto-merge feature automatically merges a PR once all required checks and reviews pass. Surfacing this status in the dashboard helps users track which PRs will auto-merge and lets them enable it for PRs that are close to ready.

## Requirements

1. **GraphQL query**: Fetch `autoMergeRequest { enabledAt mergeMethod }` on PullRequest nodes
2. **Data model**: Add `auto_merge_enabled` field and related properties to `PullRequest`
3. **Badge display**: Show "AUTO-MERGE" badge in PR list rows and detail pane when enabled
4. **Toggle keybinding**: Add keybinding to enable/disable auto-merge via `gh pr merge --auto` / `gh pr merge --disable-auto`
5. **Tests**: Test auto-merge status parsing, badge rendering, and toggle command construction
