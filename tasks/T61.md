## Objective

Fetch label hex colors from the GitHub API and render them as Rich-styled colored badges in both the PR list and detail pane.

## Background

GitHub labels have associated hex colors that help visually categorize PRs. Currently labels may be shown as plain text. Rendering them with their actual colors using Rich markup provides a more informative and visually distinctive display.

## Requirements

1. **GraphQL query**: Fetch `labels.nodes { name color }` with hex color values
2. **Data model**: Add `color` field to label representation in `PullRequest` model
3. **Rich rendering**: Convert hex colors to nearest terminal color and render labels as colored badges using Rich markup
4. **Display**: Show colored labels in both PR list rows (compact) and detail pane (full)
5. **Tests**: Test hex-to-terminal color conversion and label badge rendering
