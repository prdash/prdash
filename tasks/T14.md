## Objective

Add the ability to save an `AppConfig` to a TOML file, enabling the wizard and settings screen to persist configuration.

## Requirements

1. `save_config(config: AppConfig, path: Path | None = None) -> None`
2. Hand-format TOML with double-quoted strings, boolean literals, proper sections
3. Create parent directories if they don't exist
4. Write atomically via temp file + `os.replace()`
5. Helper `_serialize_config_toml(config) -> str` for formatting
