## Objective

Build an async GraphQL client that fetches PR data from GitHub for each of the 5 query group types, and parse responses into Pydantic data models.

## Requirements

1. Async httpx-based GraphQL client for `https://api.github.com/graphql`
2. One GraphQL query per group type (5 separate calls per refresh)
3. Pydantic data models for PR data
4. Parse GraphQL JSON responses into typed models
5. Timeline events limited to: IssueComment, PullRequestReview, HeadRefForcePushed

### Data Models (`models.py`)

```python
from datetime import datetime
from pydantic import BaseModel

class Reviewer(BaseModel):
    login: str
    status: str  # "APPROVED", "CHANGES_REQUESTED", "PENDING", "COMMENTED", "DISMISSED"

class CheckRun(BaseModel):
    name: str
    status: str       # "QUEUED", "IN_PROGRESS", "COMPLETED"
    conclusion: str | None  # "SUCCESS", "FAILURE", "NEUTRAL", etc.

class TimelineEvent(BaseModel):
    type: str          # "IssueComment", "PullRequestReview", "HeadRefForcePushed"
    author: str
    created_at: datetime
    body: str | None = None  # For comments/reviews

class PullRequest(BaseModel):
    id: str            # GraphQL node ID
    number: int
    title: str
    author: str
    url: str
    created_at: datetime
    body: str | None = None
    labels: list[str] = []
    reviewers: list[Reviewer] = []
    checks: list[CheckRun] = []
    timeline_events: list[TimelineEvent] = []

    @property
    def ci_status(self) -> str:
        """Aggregate CI status: 'passing', 'failing', 'pending', 'none'."""
        ...

    @property
    def review_status(self) -> str:
        """Aggregate review status: 'approved', 'changes_requested', 'pending', 'none'."""
        ...

class QueryGroupResult(BaseModel):
    """PRs for a single query group."""
    group_name: str
    group_type: str
    pull_requests: list[PullRequest] = []
```

### GraphQL Queries (`queries.py`)

Each query group type has its own query. All queries should request the same PR fields but use different search qualifiers:

- **direct_reviewer**: `review-requested:{username}`
- **team_reviewer**: `team-review-requested:{org}/{team_slug}` (one per team, merge results)
- **mentioned**: `mentions:{username}`
- **authored**: `author:{username}`
- **label**: `label:"{label}"` (one per label, merge results)

Common search prefix: `repo:{org}/{repo} is:pr is:open`

Example query structure:
```graphql
query($searchQuery: String!, $first: Int!, $after: String) {
  search(query: $searchQuery, type: ISSUE, first: $first, after: $after) {
    pageInfo { hasNextPage endCursor }
    nodes {
      ... on PullRequest {
        id
        number
        title
        url
        createdAt
        body
        author { login }
        labels(first: 20) { nodes { name } }
        reviewRequests(first: 20) { nodes { requestedReviewer { ... on User { login } ... on Team { slug } } } }
        reviews(last: 20) { nodes { author { login } state } }
        commits(last: 1) { nodes { commit { statusCheckRollup { contexts(first: 50) { nodes {
          ... on CheckRun { name status conclusion }
          ... on StatusContext { context state }
        } } } } } }
        timelineItems(last: 50, itemTypes: [ISSUE_COMMENT, PULL_REQUEST_REVIEW, HEAD_REF_FORCE_PUSHED_EVENT]) {
          nodes {
            ... on IssueComment { author { login } createdAt body }
            ... on PullRequestReview { author { login } createdAt body state }
            ... on HeadRefForcePushedEvent { actor { login } createdAt }
          }
        }
      }
    }
  }
}
```

### Client (`client.py`)

```python
class GitHubClient:
    def __init__(self, token: str):
        self._client = httpx.AsyncClient(
            base_url="https://api.github.com",
            headers={
                "Authorization": f"Bearer {token}",
                "Content-Type": "application/json",
            },
        )

    async def execute_query(self, query: str, variables: dict) -> dict:
        """Execute a GraphQL query and return the response data."""
        ...

    async def fetch_group(self, config: AppConfig, group: QueryGroupConfig) -> QueryGroupResult:
        """Fetch PRs for a single query group."""
        ...

    async def fetch_all_groups(self, config: AppConfig) -> list[QueryGroupResult]:
        """Fetch PRs for all enabled query groups (5 parallel calls)."""
        ...

    async def close(self):
        await self._client.aclose()
```

- Handle pagination if needed (unlikely for open PRs in a single repo, but support `after` cursor)
