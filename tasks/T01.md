## Objective

Initialize the Python project with `uv`, configure build metadata, install core dependencies, and establish the `src/` layout directory structure so all subsequent tasks have a working foundation.

## Requirements

1. Initialize project with `uv init`
2. Configure `pyproject.toml` with correct metadata
3. Create `src/gh_review_dashboard/` package with `__init__.py` and `__main__.py`
4. Install core dependencies
5. Verify the project runs via `python -m gh_review_dashboard`

## Implementation Details

### pyproject.toml

```toml
[project]
name = "gh-review-dashboard"
version = "0.1.0"
requires-python = ">=3.12"
dependencies = [
    "textual",
    "httpx",
    "pydantic",
]

[build-system]
requires = ["hatchling"]
build-backend = "hatchling.build"

[tool.hatch.build.targets.wheel]
packages = ["src/gh_review_dashboard"]
```

### Directory Structure

```
src/
  gh_review_dashboard/
    __init__.py
    __main__.py
```

### Files to Create

- **`src/gh_review_dashboard/__init__.py`**: Empty or minimal (package marker).
- **`src/gh_review_dashboard/__main__.py`**: Entry point. For now, a simple print or placeholder that confirms the app can launch:
  ```python
  def main():
      print("gh-review-dashboard is running")

  if __name__ == "__main__":
      main()
  ```

### Steps

1. Run `uv init` in the project root (if not already initialized)
2. Edit `pyproject.toml` to match the configuration above
3. Create the `src/gh_review_dashboard/` directory and files
4. Run `uv sync` to install dependencies
5. Verify with `uv run python -m gh_review_dashboard`

## Acceptance Criteria

- [ ] `pyproject.toml` exists with name `gh-review-dashboard`, `requires-python = ">=3.12"`, and dependencies `textual`, `httpx`, `pydantic`
- [ ] `src/gh_review_dashboard/__init__.py` exists
- [ ] `src/gh_review_dashboard/__main__.py` exists and is executable
