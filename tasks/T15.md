## Objective

Build a 4-step Textual Screen wizard for first-run configuration, with auto-detected defaults and per-step validation.

## Requirements

1. `WizardState` dataclass accumulates values across steps
2. `WizardStep(Screen)` base class with cancel binding (Escape exits app)
3. 4 screen subclasses:
   - `RepoScreen` — org + repo name inputs
   - `UsernameScreen` — GitHub username input
   - `TeamSlugsScreen` — checkbox list from detected teams, fallback to comma-separated text input
   - `PollIntervalScreen` — numeric input with min 30 validation
4. Final step builds `AppConfig` with `DEFAULT_QUERY_GROUPS`, calls `save_config()`, marks state completed
5. `SetupWizardApp(App)` — takes `token: str`, runs detection in a worker on mount, pushes first screen
