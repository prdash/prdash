## Objective

Add an in-app settings screen accessible via `S` keybinding that allows editing essential config fields without restarting.

## Requirements

1. `SettingsScreen(Screen[AppConfig | None])` with Input widgets pre-filled from current config
2. Fields: org, repo name, username, team_slugs (comma-separated), poll_interval
3. Validation: required fields non-empty, poll_interval >= 30
4. Save builds new `AppConfig` (preserving existing `query_groups`), calls `save_config()`, dismisses with new config
5. Cancel/Escape dismisses with `None`

### App Changes

6. `Binding("S", "settings", "Settings")` (capital S to avoid conflicts)
7. `action_settings()` pushes `SettingsScreen`, callback updates `self.config`
8. Store `self._refresh_timer` reference, cancel/recreate on config change
9. Trigger `refresh_data()` after config update
