## Objective

Show linked "Closes #123" issues in the PR detail pane using the GraphQL `closingIssuesReferences` field.

## Background

PRs often reference issues they resolve. Displaying these linked issues in the detail pane helps reviewers understand the context and purpose of a PR without navigating to GitHub.

## Requirements

1. **GraphQL query**: Add `closingIssuesReferences` field to the PR query to fetch linked issue numbers, titles, and states
2. **Data model**: Add `linked_issues` field to `PullRequest` model with issue number, title, and state
3. **Detail pane**: Render linked issues section in the detail pane with issue number, title, and open/closed state
4. **Tests**: Test GraphQL response parsing for linked issues and detail pane rendering
