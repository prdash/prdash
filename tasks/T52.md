## Objective

Add keybindings to copy PR URL or PR reference to the system clipboard for quick sharing.

## Background

Users frequently need to share PR links in chat or commit messages. Providing `y` to copy the full URL and `Y` to copy the short reference (`org/repo#123`) saves context-switching to the browser. Platform clipboard access uses `pbcopy` on macOS and `xclip`/`xsel` on Linux via subprocess.

## Requirements

1. **`y` keybinding**: Copy the selected PR's full GitHub URL to the system clipboard
2. **`Y` keybinding**: Copy the PR reference in `org/repo#123` format to the clipboard
3. **Clipboard subprocess**: Use `pbcopy` on macOS, `xclip -selection clipboard` on Linux; handle missing tools gracefully
4. **Toast confirmation**: Show a toast notification confirming what was copied (e.g., "Copied: org/repo#123")
5. **Tests**: Unit tests for clipboard command selection per platform, reference formatting, and error handling when clipboard tool is unavailable
