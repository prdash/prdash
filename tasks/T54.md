## Objective

Show a PR's diff inline in the detail pane or a dedicated diff pane, using `gh pr diff` with optional `delta` syntax highlighting.

## Background

Reviewing diffs without leaving the dashboard reduces context switching. `gh pr diff` outputs unified diff format which can be displayed directly or piped through `delta` for syntax-highlighted output. The diff view should be scrollable and dismissible.

## Requirements

1. **Diff keybinding**: Keybinding to fetch and display the diff for the selected PR via `gh pr diff <url>`
2. **Syntax highlighting**: Optionally pipe through `delta` if available on the system; fall back to raw diff
3. **Scrollable view**: Display diff in a scrollable pane with line-wrap support
4. **Dismiss**: Allow closing the diff view to return to normal detail pane
5. **Tests**: Test diff fetching, delta detection, and pane toggle behavior
