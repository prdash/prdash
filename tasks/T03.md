## Objective

Extract the GitHub API token from the `gh` CLI and validate it works, providing clear error messages when authentication is unavailable.

## Requirements

1. Extract token by running `gh auth token` as a subprocess
2. Validate the token with a simple GitHub API call
3. Provide clear error messages for common failure modes

## Implementation Details

### Files to Create

- **`src/gh_review_dashboard/auth.py`**

### Token Extraction

```python
import subprocess

def get_github_token() -> str:
    """Extract GitHub token from gh CLI."""
    try:
        result = subprocess.run(
            ["gh", "auth", "token"],
            capture_output=True,
            text=True,
            check=True,
        )
        token = result.stdout.strip()
        if not token:
            raise AuthError("gh auth token returned empty result. Run 'gh auth login' first.")
        return token
    except FileNotFoundError:
        raise AuthError("GitHub CLI (gh) not found. Install it from https://cli.github.com/")
    except subprocess.CalledProcessError:
        raise AuthError("Not authenticated with GitHub CLI. Run 'gh auth login' first.")
```

### Token Validation

```python
import httpx

async def validate_token(token: str) -> str:
    """Validate token by calling GET /user. Returns the username."""
    async with httpx.AsyncClient() as client:
        response = await client.get(
            "https://api.github.com/user",
            headers={
                "Authorization": f"Bearer {token}",
                "Accept": "application/vnd.github+json",
            },
        )
        if response.status_code == 401:
            raise AuthError("GitHub token is invalid or expired. Run 'gh auth login' to re-authenticate.")
        response.raise_for_status()
        return response.json()["login"]
```

### Custom Exception

```python
class AuthError(Exception):
    """Raised when GitHub authentication fails."""
    pass
```

## Acceptance Criteria

- [ ] `get_github_token()` returns a valid token string when `gh` is installed and authenticated
- [ ] `get_github_token()` raises `AuthError` with a clear message when `gh` is not installed
- [ ] `get_github_token()` raises `AuthError` with a clear message when `gh` is not authenticated
- [ ] `validate_token()` succeeds with a valid token and returns the username
- [ ] `validate_token()` raises `AuthError` with a clear message for an invalid/expired token
