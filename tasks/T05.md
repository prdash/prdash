## Objective

Create the Textual App subclass with a CSS-based split-pane layout, header, footer, and placeholder widgets for each pane.

## Requirements

1. Textual `App` subclass as the application entry point
2. Split-pane layout: left pane ~40%, right pane ~60%
3. Header with app title
4. Footer with keybinding hints
5. Placeholder widgets in each pane
6. Wire `__main__.py` to launch the app

## Implementation Details

### Files to Create

- **`src/gh_review_dashboard/app.py`** - App class
- **`src/gh_review_dashboard/app.tcss`** - Textual CSS

### Files to Modify

- **`src/gh_review_dashboard/__main__.py`** - Launch the app

### App Class (`app.py`)

```python
from textual.app import App, ComposeResult
from textual.widgets import Header, Footer, Static

class ReviewDashboardApp(App):
    """GitHub Review Dashboard TUI application."""

    CSS_PATH = "app.tcss"
    TITLE = "GitHub Review Dashboard"

    def compose(self) -> ComposeResult:
        yield Header()
        with Horizontal():
            yield Static("PR List (placeholder)", id="pr-list-pane")
            yield Static("Detail Pane (placeholder)", id="detail-pane")
        yield Footer()
```

### CSS Layout (`app.tcss`)

```css
#pr-list-pane {
    width: 40%;
    height: 100%;
    border-right: solid $accent;
}

#detail-pane {
    width: 60%;
    height: 100%;
}
```

### Entry Point (`__main__.py`)

```python
from gh_review_dashboard.app import ReviewDashboardApp

def main():
    app = ReviewDashboardApp()
    app.run()

if __name__ == "__main__":
    main()
```
