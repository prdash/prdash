## Objective

Create a comprehensive README documenting installation, configuration, usage, and keybindings so that a new user can set up and use the dashboard from scratch.

## Requirements

1. Installation instructions (uv-based)
2. Prerequisites (Python 3.12+, GitHub CLI installed and authenticated)
3. Configuration guide with all TOML fields documented
4. Usage guide (how to launch, what to expect)
5. Keybindings reference table

## Implementation Details

### Files to Create

- **`README.md`** in the project root

### Sections

#### 1. Title & Overview
- Project name and one-line description
- What the dashboard does and who it's for

#### 2. Prerequisites
- Python 3.12+
- `uv` package manager
- GitHub CLI (`gh`) installed and authenticated (`gh auth login`)

#### 3. Installation
```bash
git clone <repo-url>
cd GithubReviewDashboard
uv sync
```

#### 4. Configuration
- Config file location: `~/.config/gh-review-dashboard/config.toml`
- Full example config with all fields
- Field reference table:

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `repo.org` | string | required | GitHub organization or owner |
| `repo.name` | string | required | Repository name |
| `username` | string | required | Your GitHub username |
| `team_slugs` | list[string] | `[]` | Team slugs for team review queries |
| `poll_interval` | int | `300` | Auto-refresh interval in seconds |
| `query_groups` | list | (5 defaults) | PR query group configurations |

- Query group configuration with type/name/labels/enabled fields
- Available group types: `direct_reviewer`, `team_reviewer`, `mentioned`, `authored`, `label`

#### 5. Usage
```bash
uv run python -m gh_review_dashboard
```
- Brief description of the layout (left pane = PR list, right pane = detail view)
- What each section shows

#### 6. Keybindings

| Key | Action |
|-----|--------|
| `j` / `Down` | Move cursor down |
| `k` / `Up` | Move cursor up |
| `Enter` | Open PR in browser / Toggle group |
| `Tab` | Switch pane focus |
| `r` / `R` | Manual refresh |
| `q` | Quit |

#### 7. Query Groups
- Explain each of the 5 group types
- How to customize, enable/disable, add label-based groups
