## Objective

Add a `ModalScreen` triggered by `?` showing all keybindings organized by context (Navigation, Actions, App) in a formatted table.

## Background

New users need a discoverable way to learn all available keybindings without leaving the app. A `?` key is a common convention in terminal applications for showing help. Textual's `ModalScreen` provides a semi-transparent overlay that can display content above the main app.

## Requirements

1. **Help screen**: Create `HelpScreen(ModalScreen)` in `screens/help.py`
2. **Keybinding groups**: Organize shortcuts into sections: Navigation (j/k, g/G, Tab, arrow keys), Actions (Enter/o, c, r), App (?, S, Ctrl+P, q)
3. **Layout**: Display keybindings as a formatted table with Key and Description columns, grouped under section headers
4. **CSS overlay**: Semi-transparent background overlay with a centered content container
5. **Dismissal**: Escape or `?` again dismisses the overlay
6. **App binding**: Add `?` binding to `App.BINDINGS` that pushes `HelpScreen`
7. **Re-exports**: Update `screens/__init__.py` to export `HelpScreen`
8. **Tests**: Verify screen mounts, displays expected keybindings, and dismisses on Escape
