## Objective

Build the right-pane widget that displays detailed information about the currently selected PR, including metadata, description, labels, reviewers, CI checks, and event timeline.

## Requirements

1. Show full PR details when a PR is selected
2. Display an empty/placeholder state when no PR is selected
3. Sections: metadata summary, description, labels, reviewers, CI checks, timeline
4. Replace the right-pane placeholder from T05

### Empty State

When no PR is selected:
```
  Select a pull request from the list
  to view its details.
```

### Event Wiring

In `app.py`, handle the `PRSelected` message:
```python
def on_pr_selected(self, event: PRSelected) -> None:
    detail_pane = self.query_one(DetailPaneWidget)
    detail_pane.show_pr(event.pull_request)
```

## Acceptance Criteria

- [ ] Widget shows empty state message when no PR is selected

- [ ] Labels are listed (or "None" if empty)

- [ ] Timeline shows events in chronological order with type, author, time, and body
