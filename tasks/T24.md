# T24: CLI Entry Point (`ghrd` command)

## Summary

Add a `[project.scripts]` entry point so the app can be run as `ghrd` instead of `uv run python -m gh_review_dashboard`. Enables global installation via `uv tool install`.

## Dependencies

- T01 (project setup)

## Changes

1. **`pyproject.toml`** — Add `[project.scripts]` section mapping `ghrd` to `gh_review_dashboard.__main__:main`
2. **`README.md`** — Add global install instructions (`uv tool install`), update Quick Start and Usage to use `ghrd`
3. **`CLAUDE.md`** — Update Commands section with `ghrd` as primary run command
4. **`WORK_TRACKER.md`** — Add Phase 9: Distribution with T24 row

## Verification

1. `uv sync` picks up the new entry point
2. `uv run ghrd` launches the dashboard
3. `uv tool install .` from project root, then `ghrd` from any directory
