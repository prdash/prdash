## Objective

Implement keyboard navigation for the PR list, group collapse/expand, and focus switching between panes.

## Requirements

1. `j`/`k` and arrow keys navigate up/down in the PR list
2. `Enter` on a group header toggles collapse/expand
3. `Tab` (or similar) switches focus between left and right panes
4. Navigation skips collapsed group contents
5. Footer shows keybinding hints

## Implementation Details

### Files to Modify

- **`src/gh_review_dashboard/app.py`** - Bind keys, manage focus
- **`src/gh_review_dashboard/widgets/pr_list.py`** - Handle navigation within the list

### Keybinding Registration

In `app.py`:
```python
BINDINGS = [
    Binding("q", "quit", "Quit"),
    Binding("tab", "switch_pane", "Switch Pane"),
    Binding("j", "cursor_down", "Down", show=False),
    Binding("k", "cursor_up", "Up", show=False),
]
```

### PR List Navigation

The PR list widget should maintain a cursor index. Navigation keys move the cursor:

- **Down** (`j`, `down`): Move to next visible row (PR or group header)
- **Up** (`k`, `up`): Move to previous visible row
- **Enter** on group header: Toggle collapse/expand for that group
- Skip over hidden rows (PRs in collapsed groups)

When cursor moves to a PR row, emit `PRSelected` to update the detail pane.

### Focus Switching

- `Tab` cycles focus between the PR list pane and the detail pane
- The focused pane should have a visual indicator (e.g., highlighted border)
- When detail pane is focused, allow scrolling its content with `j`/`k` or arrows

### Visual Feedback

- Highlighted cursor row in the PR list (distinct background color)
- Focused pane border styling in CSS:
  ```css
  #pr-list-pane:focus-within {
      border: solid $accent;
  }
  ```
