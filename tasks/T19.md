## Objective

Dedicated screen for managing query groups in-app: add, remove, reorder, and toggle enabled state.

## Requirements

1. Dynamic list UI showing all query groups with per-group sub-fields (type, name, labels, enabled)
2. Add/remove/reorder operations
3. Accessible from settings screen or separate keybinding
4. Save persists to TOML config
