## Objective

Show deployment environment status (e.g., Vercel or Netlify preview URLs) in the PR detail pane via the GraphQL deployments field.

## Background

Many teams use preview deployments for PRs. Showing deployment status and URLs directly in the dashboard lets reviewers quickly access preview environments without navigating to GitHub's checks tab.

## Requirements

1. **GraphQL query**: Fetch deployment data via `commits.nodes.commit.deployments` or equivalent field
2. **Data model**: Add `Deployment` model with environment name, state, and URL
3. **Detail pane**: Render deployment status section showing environment name, state (active/pending/failed), and preview URL
4. **Open URL**: Allow opening the deployment URL in a browser via keybinding
5. **Tests**: Test deployment data parsing and detail pane rendering
