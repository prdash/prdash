## Objective

Integrate the setup wizard into the app entry point so it launches automatically when no config file exists.

## Requirements

1. Move `get_github_token()` before `load_config()` (token needed for wizard detection)
2. If config file doesn't exist, run `SetupWizardApp(token=token).run()`
3. If wizard exits without completing (user cancelled), exit gracefully with code 0
4. If wizard completes, proceed with normal `load_config()` -> dashboard flow
