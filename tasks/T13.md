## Objective

Provide defensive auto-detection functions that discover org/repo, username, and team slugs for pre-filling the setup wizard.

## Requirements

1. `detect_repo_from_git_remote() -> tuple[str, str] | None` — parse `git remote get-url origin` for SSH and HTTPS GitHub URLs
2. `detect_username(token: str) -> str | None` — call `validate_token()` wrapped in try/except
3. `detect_team_slugs(token: str, org: str, username: str) -> list[str]` — `GET /user/teams`, filter by org (case-insensitive)
4. All functions return `None` or empty list on failure — never raise
